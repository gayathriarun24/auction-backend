# Auction Platform - Backend

This is the Node.js/Express backend API for the Auction Platform. It handles user authentication, product CRUD operations, auction logic, and real-time bid processing.

## 🚀 Features

* **Authentication**: Secure JWT-based user registration and login.
* **Product Management**: RESTful API endpoints for creating, reading, updating, and deleting auction items.
* **Auction Logic**: Handles various auction types and bidding processes.
* **Notification Service**: Integrated email service (Brevo API) to alert users when outbid.
* **Inventory Control**: Management of unsold items and auction statuses.

## 🛠 Tech Stack

* **Runtime**: Node.js
* **Framework**: Express.js
* **Database**: MongoDB (Mongoose)
* **Email Service**: Brevo API
* **Security**: JWT (JSON Web Tokens), bcrypt

## 📋 Getting Started

### Prerequisites
* Node.js (v24.15.0 or higher)
* MongoDB database connection string
